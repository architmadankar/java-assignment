# RIDE-SHARING CHARGES ESTIMATOR AND COMPARISON SYSTEM

## How to Run 

### Requirements JAVA 8 
How to run 
```Bash
    git clone https://github.com/architmadankar/java-assignment.git
    cd java-assignment
```
In Command Prompt 
```Bash
  javac Comparision.java
  java Comparision
```
